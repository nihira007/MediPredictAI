About

✖

MediPredictAI is an AI-powered disease prediction and medical assistance platform designed to analyze symptoms and provide possible conditions along with medical suggestions.


Using a trained machine learning model, the system evaluates user-selected symptoms to predict possible diseases along with confidence scores, severity levels, and recommended precautions—helping users make informed health decisions.


Key Features:

  • Multi-disease prediction with probability ranking using Machine Learning.
  • Severity analysis with personalized precautions & recommendations.
  • AI-powered assistant for real-time health queries.


How It Works:
Select symptoms → ML model analyzes patterns → Predictions are generated → Results are enhanced with medical insights and AI guidance.


Built with: Python, FastAPI, Machine Learning-Random Forest (Tuned), AI Assistant (Groq)
